# Extensions for Angular developers @ Trivadis
Useful VS Code extensions for any Angular developer.

## File settings

```json
    "files.exclude": {
        "**/.git": true,
		"**/.DS_Store": true,
        "**/*.js.map": true,
        "**/app/**/*.js": true,
        "**/e2e/**/*.js": true
    },
    "files.defaultLanguage": "typescript",
	"files.hotExit": "off",
	"workbench.iconTheme": "vscode-icon-theme",
```
